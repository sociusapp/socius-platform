# Product features (mobile)

Large user flows live under **`src/features/`**, separate from generic screens like Auth or Home shell.

| Feature | Path | What it is |
|--------|------|------------|
| **Daily Help** | `daily-help/` | Local help requests — Community tab, requester + helper (volunteer) flows |
| **Need Presence** | `need-presence/` | Awareness / presence requests — requester + responder (witness) flows |

Shared app code stays in `src/components/`, `src/services/`, `src/navigation/`, etc.

Each feature folder has its own `README.md` with screen layout.
